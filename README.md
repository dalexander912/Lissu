# Lissu

**Lissu** es una aplicación móvil que te permite gestionar listas y recordatorios de compras.

La aplicación cuenta con las siguientes funcionalidades principales:

* Registro y autenticación de usuarios
* Gestión de listas de compras
* Recordatorios para productos seleccionados
* Listas de supermercados cercanos
* Escaneo de códigos de barra

## 🛈 Sobre Lissu

* Versión: 1.0
* Sistema requerido: Android 7.0 (Nougat) o superior
* Tecnologías:
    * App: Kotlin, Jetpack Compose, Material Design 3
    * Backend: JavaScript, Express

## 🌐 APIs externas consumidas

| API             | Uso                                |
| --------------- | ---------------------------------- |
| Google ML Kit   | Lectura de códigos de barra        |
| Open Food Facts | Registro de datos de productos     |
| Geoapify Places | Búsqueda de supermercados cercanos |
